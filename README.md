# 🏆 Inter-House Tournament Tracker

A web application designed to **track and display inter-house tournament results**.
Built for school use, it allows students and viewers to easily follow match outcomes, standings, and player performance.

---

## 📌 Overview

This project was developed to simplify how inter-house tournaments are managed and viewed.
Instead of relying on manual updates or announcements, this system provides a **centralized, digital scoreboard** accessible to everyone.

---

## ✨ Features

- ⚽ **Futsal & Handball Support**
- 📊 **Live Score Updates**
- 🏆 **Automatic Standings Table**
- 🎯 **Player Goal Tracking** _(currently only for futsal)_
- 📅 **Match Results Display**
- 📱 **Responsive Design (Mobile + Desktop)**
- 🎨 **Modern UI with Animated Interface** _(ongoing improvements)_
